Frequently asked questions

How do I set up my first room?

Tap Add Room, name the room, then add steps in play order — one physical action per step, each with a reference photo of the correct reset state. If your team resets back-to-front, set Run order to Reverse and the checklist will run your steps backwards.

What is the double-check pass?

After every step is checked once, the app requires a second sweep: each step is shown again, photo first, for confirmation. Only then can the reset be signed off. This is deliberate — the second look is what catches the missed padlock.

What does the free version include?

One fully functional room — steps, photos, timer, double-check, incidents, history, and exports. The one-time Reset Pro purchase ($49.99) unlocks unlimited rooms on that device. No subscription.

How do I move my data to a new device?

Settings → Export Backup creates a single file with all rooms, photos, staff, and history. On the new device (fresh install), use Settings → Import Backup. The app also rides along in normal device backups.

Does it work offline?

Completely. The app never needs a network connection — nothing is stored in the cloud.

Can I use it in Guided Access / kiosk mode?

Yes — the app is designed for a shared device at the game master station. All flows work inside Guided Access, and the run screen has no destructive single-tap actions.

My purchase didn't unlock. What do I do?

Settings → Restore Purchases while signed into the same Apple Account that bought Reset Pro. If it still doesn't unlock, email us.
